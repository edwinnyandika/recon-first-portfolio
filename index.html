About Me

I FIND WHAT
OTHERS MISS.

I'm Edwin Nyandika — a self-taught security researcher and CS/IT student based in Nairobi, Kenya. I don't specialize in one lane. I go wide, then ruthlessly deep.

My workflow starts where most hunters stop: deep automated recon using custom Python pipelines — Subfinder, Amass, httpx — mapping the full attack surface before a single payload fires.

I take signal seriously. A 3.0 on HackerOne beats a trophy shelf of noise. Every report I submit is built to be accepted, not just reviewed. Clear PoC, CVSS score, and a remediation path included.

Beyond bug bounty, I've built platforms with 10k+ users — which means I understand both the attacker and the builder perspective. That dual lens is rare. I use it.
